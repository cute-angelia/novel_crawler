## 功能

爬取网站上小说章节的内容，并以txt形式保存在本地。

程序运行需要的参数如下：

| 参数名 | 参数值     | 样例                                |
|-----|---------|-----------------------------------|
| -u  | 小说列表的链接 | https://www.22biqu.com/biqu51722/ |
| -f  | 文件名（可选） | 斗破苍穹                              |
| -n  | 多少章（可选） | 0-100                             |
 
复制一份 config.toml.example 改名 为 config.toml

其中robotkey是企业微信机器人，下载完成后通知用。

示例：

```bash
go run main.go -u https://www.22biqu.com/biqu5735/
```
